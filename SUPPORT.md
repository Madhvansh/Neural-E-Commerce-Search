# Support

This project is maintained as alpha research software and has no guaranteed
response time or production support agreement.

- Use a bug report for a minimal, reproducible defect.
- Use a feature request for a focused use case and evaluation plan.
- Use a reproduction report when you have rerun or challenged a published
  experiment.
- Follow [SECURITY.md](SECURITY.md) for vulnerabilities or sensitive reports.

Include the version or commit, operating system, Python version, installed
extras, exact command, smallest legal sample input, observed output, and expected
output. Remove secrets, customer data, private datasets, and model credentials.

Questions about adapting the system to a private catalogue may be discussed at
the architectural level, but maintainers cannot inspect proprietary data or
guarantee relevance, latency, safety, or business outcomes.
